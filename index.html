Vocifery

Vocifery is an Internet-based question answering (QA) engine. It will parse a question and send queries to multiple content sources such as Wikipedia and search engines. The engine then aggregates the results and searches for candidate answers.

Issues

  • Can only handle trivial questions, such as "who assassinated Abraham Lincoln?"
  • Many erroneous answers retrieved (false positives)
  • Many correct answers not retrieved (false negatives)
  • Poor system responsiveness
  • Does not scale for worldwide Internet users

Goals

  • Incorporate more question types
  • Improve precision and recall
  • Chain together facts to find answers by inference
  • Move QA engine to better for servers for improved performance
  • Optimize QA engine for scalability
